# ReadInt Basic

This example demonstrates integer input with `ReadInt()`.

## What it shows

- reading one integer token from stdin
- assigning the result to a variable
- printing the value using `WriteInt` and `WriteLn`

## Source

- `src/Main.ob0`

## Run

From the repository root:

```bash
printf "42\n" | scripts/oberon0 examples/readint-basic --run
```

## Expected output

```text
42
```
