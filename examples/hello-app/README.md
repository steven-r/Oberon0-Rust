# Hello App

This example shows the smallest useful Oberon0 program:

- arithmetic expressions
- assignment
- procedure-style output call (`WriteInt`)
- optional dependency manifest file (`oberon.toml`)

## Source

- `src/Main.ob0`
- `oberon.toml`

## Run

From the repository root:

```bash
scripts/oberon0 examples/hello-app --run
```

## Expected output

```text
7
State: {"x": 7}
```
