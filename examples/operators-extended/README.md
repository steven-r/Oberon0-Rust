# operators-extended

This example demonstrates the extended operator surface from issue #18:

- unary sign operators (`+`, `-`)
- integer operators (`DIV`, `MOD`)
- boolean operators (`OR`, `&`, `~`)

Run it from the repository root:

```bash
scripts/oberon0 examples/operators-extended --run
```

Expected output:

- first line: computed integer result
- second line: boolean result encoded as `0` or `1`
